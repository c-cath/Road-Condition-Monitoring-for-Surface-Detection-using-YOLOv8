
# RoadWatch

The **RoadWatch** system is an AI-powered road condition monitoring application designed to improve the safety and efficiency of road infrastructure maintenance. It uses Artificial Intelligence (AI) and Computer Vision to detect and classify road surface defects in real time, such as cracks, potholes, and other anomalies.

---

## Features

- Real-time road defect detection using the YOLOv8 object detection model.  
- Identifies cracks, potholes, and other surface irregularities.  
- Built with Python libraries including OpenCV, Winsound, and Tkinter for real-time processing and user interface.  
- Trained on a large dataset of 5,718 images containing 9,410 labeled road defect instances.
